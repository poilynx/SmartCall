#特性：  
1. 如果同事不在你本地通讯录，那么他给你打电话的时候，会自动显示其头像、名字、职位  
2. 支持一键将同事的头像、公司、部门、职位、家庭电话、公司电话添加到本地通讯录  
3. 使用不耗费任何流量  
4. 使用非常简单，短按拨打号码、长按查看详情、下拉刷新数据
